HTML Code Preview

\n

To preview some HTML code without it being rendered, you can use HTML entities or the <pre> and <code> tags.

\n \n

Example using HTML Entities:

\n 
\n        <div>This is a <strong>bold</strong> text.</div>\n
\n \n

Example using <pre> and <code> tags:

\n 
\n<div>This is a <strong>bold</strong> text.</div>\n
\n\n
